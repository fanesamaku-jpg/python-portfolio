# Data Analysis with Python

Projects completed as part of freeCodeCamp's Data Analysis with Python certification.

## Projects

1. Mean-Variance-Standard Deviation Calculator - NumPy statistical operations
2. Demographic Data Analyzed - Pandas data manipulation and aggregation
3. Medical Data Visualizer - Correlation analysis and categorical plotting
4. Page View Time Series Visualizer - Temporal data and trend analysis
5. Sea Level Predictor - Linear regression and forecasting

Skills demonstrated: Pandas, NumPy, Matplotlib, Seaborn, data cleaning, statistical analysis, data visualization, exploratory data analysis

Libraries used: pandas, numpy, matplotlib, seaborn, scipy
